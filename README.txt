# Демонстрация автотестирования на Python с FastAPI

Этот проект представляет собой простой веб-сервер на Python с использованием FastAPI, который демонстрирует возможности автотестирования и управления UI-элементами через REST API. В проекте реализованы:

- Управление отображением UI-элементов (кнопки, панели, combobox, dropdown) через REST API с поддержкой всех основных HTTP методов (GET, POST, PUT, PATCH, DELETE).
- Веб-интерфейс для настройки и отображения UI-элементов.
- Запуск автотестов с выбором типа (UI тесты, API тесты, все тесты).
- Логирование выполнения автотестов с отображением в веб-интерфейсе.
- Отдельный интерфейс для запуска pytest автотестов и REST клиента для вызова произвольных REST операций.
- Запуск проекта через Docker.

---

## Структура проекта

- `main.py` — основной сервер с REST API и веб-интерфейсами.
- `test_autotests.py` — пример автотестов с использованием pytest и httpx.

## Установка и запуск

### Локально

uvicorn main:app --host 0.0.0.0 --port 8000

Через Docker:
docker build -t autotest-demo .
docker run -p 8000:8000 autotest-demo

http://localhost:8000 — основной UI для управления элементами и запуска автотестов.
http://localhost:8000/logs — интерфейс для запуска pytest и REST клиента.
