Проект демонстрирует навыки QA Automation и Системного анализа. Это веб-приложение, которое объединяет:
Backend (FastAPI): Реализация REST API (GET/PUT) для управления состоянием UI.

# 🛠 Full-Stack QA Automation Sandbox

Этот проект демонстрирует компетенции на стыке **Системного анализа** и **QA Automation (Python)**. 
Я реализовал среду, которая сама себя тестирует, объединяя Backend, Frontend и инфраструктуру тестирования.

## 🚀 Технологический стек
- **Backend:** FastAPI (Python) — реализация REST API (GET/PUT).
- **Frontend:** Streamlit — динамический UI, управляемый через API.
- **Contract Testing:** Pydantic — строгая валидация JSON-схем ответов сервера.
- **E2E Automation:** Playwright — кроссбраузерное тестирование UI со скриншотами.
- **DevOps:** Docker — контейнеризация всего решения.

## 💡 Ключевые фичи
- **Интерактивность:** Вы меняете состояние кнопки через API-панель, и автотест проверяет, изменилась ли она в DOM.
- **Скриншоты в реальном времени:** При запуске тестов Playwright делает снимок экрана и выводит его в Dashboard.
- **Валидация схем:** Система упадет с подробным логом, если API нарушит контракт данных.

## 📦 Как запустить локально
1. `docker build -t qa-sandbox .`
2. `docker run -p 8501:8501 -p 8000:8000 qa-sandbox`
3. Откройте `http://localhost:8501`
